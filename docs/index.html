About



Snips & Clips



Python-based video editing program that automates removal of bad takes and creates a compilation of good takes given user-defined video files and take-start and take-end words.


Dependencies

  • Python3
  • ffmpeg
    • Download from: https://ffmpeg.org/download.html
    • Make a note of the location of the executable/binary
  • ibm-watson
    • Download using the requirements.txt file provided in the package
